# Анализатор данных сенсора температуры

**Автор:** Кирилл Тюшин, СКБ252
*tg:* @KiaRio117

## Описание
Программа для анализа показаний датчика температуры, считывающего значения в диапазоне от -50 до +50°C.

## Содержимое
- **main.cpp** - основной исходный код программы анализатора сенсора
- **in.txt** - шаблон файла для ввода данных (первое число - количество измерений, затем значения температуры)

## Функциональность
- Считывание измерений в динамический массив
- Расчет статистики: среднее значение, минимум, максимум
- Фильтрация значений, превышающих среднее
- Поддержка ввода с клавиатуры и из файла

## Использование

### Ввод с клавиатуры:
```bash
./program
```

### Чтение из файла:
```bash
./program filename.txt
```

## Формат данных
- Первое число - количество измерений (N)
- Последующие N чисел - значения температуры от -50 до +50°C

## Пример
```
=== Анализ сенсора ===
Введите количество измерений: 6
Введите значения: 10 15 8 -2 18 20

Среднее значение: 11.5
Минимум: -2
Максимум: 20

Значения выше среднего:
15.0 18.0 20.0
```
